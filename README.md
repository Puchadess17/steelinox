## 🛠️ Instalación del entorno de desarrollo

1. Asegúrate de tener **XAMPP** instalado y con el servicio de **MySQL iniciado**.
2. Clona este repositorio en tu carpeta pública (`htdocs` o equivalente).
3. Entra en la carpeta raíz del proyecto y ejecuta el script de base de datos según tu sistema operativo:

**🖥️ Para usuarios de Windows:**
Haz doble clic en el archivo `instalar_bd.bat`.

**🐧 Para usuarios de Linux / 🍎 Mac:**
Abre la terminal en la carpeta del proyecto, dale permisos de ejecución al script y lánzalo:
```bash
chmod +x instalar_bd.sh
./instalar_bd.sh
